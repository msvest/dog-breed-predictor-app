About

This web app implements a CNN model to predict dog breed based on the user-supplied image. To greet the user properly, a pre-trained OpenCV face detector is used to check whether the image contains a human, and ResNet-50 is used to detect whether the image contains a dog.

For further details, see the GitHub page for this web app, or the GitHub page for the development of these models.

Privacy

The images uploaded by the user are only stored in volatile memory, and are only used for the purposes of predicting the dog breed. No image is permanently saved into storage.
